# Famous Places

Prosty projekt edukacyjny prezentujący słynne miejsca z całego świata.  
Projekt pozwala na przeglądanie lokalizacji, filtrowanie według regionu oraz zmianę języka strony (polski, ukraiński, rosyjski, angielski).  
Stworzony w celu nauki podstaw tworzenia stron internetowych: HTML, CSS i JavaScript.

## 🌍 Demo

**Zobacz na żywo:**  
[https://biletskyikyrylo.github.io/famouspalces/](https://biletskyikyrylo.github.io/famouspalces/)

## ✨ Funkcje

- Galeria słynnych miejsc ze zdjęciami i opisami
- Filtrowanie po regionie (Wszystkie, Europa, Azja, Ameryka, Afryka, Oceania)
- Wsparcie wielu języków (PL / UA / RU / EN)
- Przełącznik trybu ciemnego
- Podstawowa responsywność (w trakcie rozwoju)

## 🚀 Jak uruchomić

Aby uruchomić projekt lokalnie:

1. Sklonuj repozytorium:
   git clone https://github.com/BiletskyiKyrylo/famouspalces.git

2. Otwórz plik `index.html` w przeglądarce.

Nie są wymagane żadne dodatkowe biblioteki ani narzędzia budujące.
